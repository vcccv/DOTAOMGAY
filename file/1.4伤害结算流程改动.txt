"火焰气息" 和 "集中火力" 现在为特效伤害调整, 影响基础攻击力与额外攻击力。
"腐尸毒" 现在不再会对友军和魔法免疫单位生效。

传统伤害调整Ⅱ
{
    ①：物理伤害无效化，影响全性质物理伤害
    {
        极寒之拥
        守护天使
        虚无
    }
    ②：魔法抗性伤害调整，影响全性质魔法伤害
    ③：幻象特殊伤害调整，影响全性质攻击伤害
    当幻象在攻击肉山时，其攻击造成的伤害将降低80%。
    当幻象在攻击建筑物并且未触发偷塔保护时，其攻击造成的伤害将降低60%。
    当幻象在攻击建筑物并且触发偷塔保护时，其攻击造成的伤害将降低75%。
}

召唤物对建筑伤害减少，影响全类型全性质伤害
六技能模式下 一些召唤物对防御塔的伤害将会被降低

核心伤害调整
{
    ①：灵能之刃飞溅伤害。本小类伤害的原始数值取决于先前造成伤害的值，即结算至此时的值。本小类的伤害仍需从头进行伤害结算流程。
    ②：攻击吸血的基础值通常为结算至此时的值。
    上述部分调整仍可能不会影响攻击吸血的基础值。
    ③：伤害承受方，影响全类型全性质伤害，加法叠加
    {

        折射
        奔袭冲撞
        血肉傀儡
        寒冬诅咒
        过载
        破碎暗影步
    }
}

尾端伤害调整
本小类默认影响全类型全性质伤害。
当一个单位处于多个这些调整状态下，且同时满足触发条件，这些技能的其他效果也会同时触发，包括且不限于造成眩晕、记录治疗数值、增减计数器等。
{
    折光
    尖刺外壳
    回到过去
    虚妄诺言
    回光返照
}

末端伤害格挡
本小类默认影响全类型全性质伤害。
本小类独立计数且仅最高值生效
{
    陵卫斗篷
    无光之盾
    船油
}

技能吸血
技能吸血的基础值通常为结算至此时的值。
幻象承受伤害调整
幻象受到任意伤害时，会受到一个伤害调整效果。

清零区
如果结算至此处，伤害的数值仍为负值，那么伤害将会更正为0。
